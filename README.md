# pi-packages

## 安装

```bash
pi install git:github.com/serosme/pi-packages
```

## 技能

- **code-review** — 通过分析暂存区更改，检测潜在问题并提供改进建议
- **commit** — 通过分析暂存区更改，生成 Conventional Commits 风格的提交信息
- **md-lint** — 使用 markdownlint-cli2 检测并自动修复 Markdown 文件格式

## 扩展

- **llm-write-guard** — 写操作守卫，拦截项目目录之外的文件写入
